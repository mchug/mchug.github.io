Чому прохідні бали подані у відсотках?

На жаль, максимально можливі бали при вступі останні роки часто змінюються.
Тому ми вирішили загостирити вашу увагу не на цифрах, а саме на результаті ЗНО.


Результатом ми називаємо одержані тобою бали під час тестування + атестат + додаткові бали й поділені на максимально можливі бали.


У 2012 та 2013 роках максимальний бал при вступі до КПІ міг сягати 870 на всі напрями (окрім напряму Образотворче мистецтво (6.020205), для нього максимальний бал становив 1250).


Формула: 870 = 200 балів * три ЗНО + 200 балів за атестат + 50 балів МАН/ВУО + 20 балів підготовчі курси КПІ

Тобто: 870 = 200*4 + 50 + 20


У 2014 році максимальний бал для технічних напрямів становив 680, для напряму Образотворче мистецтво - 1110 та для нетехнічних напрямів - 710.


Формула для технічних напрямів: 680 = 200 балів * два ЗНО + 200 балів по ЗНО з української мови та літератури * 0.75 + 60 балів за атестат + 50 балів МАН/ВУО + 20 балів підготовчі курси КПІ

Тобто: 680 = 200*2 + 150 + 60 + 50 + 20


Формула для нетехнічних напрямів: 710 = 200 балів * три ЗНО + 60 балів за атестат + 50 балів МАН/ВУО

Тобто: 710 = 200*3 + 60 + 50


При підрахунку балів для напряму Образотворче мистецтво треба також враховувати творчі конкурси.
